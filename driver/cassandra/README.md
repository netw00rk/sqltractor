# Cassandra Driver

## Usage

```bash
sqltractor-cli -url cassandra://host:port/keyspace -path ./db/migrations create add_field_to_table
sqltractor-cli -url cassandra://host:port/keyspace -path ./db/migrations up
sqltractor-cli help # for more info
```
